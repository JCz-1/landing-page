# Landing Page

A responsive landing page featuring a welcome screen that links to the main homepage, with a navigation bar including a login link to a functional form page. Built with HTML and CSS.

## Features
- Welcome page linking to main page
- Navigation bar with login link
- Login form page
- Fully styled with CSS

## How to View
1. Open `index.html` in your browser
2. Navigate using the links in the navbar

## Technologies
- HTML
- CSS
